# Standard Library functions

Extron comes with various built-in functions that are available by importing them.This page provides an overview of the standard library functions available in Extron.

They can be imported by 

```ts twoslash
import "std:math"
//     ^^^^^^^^^^ can be any of the following
// @annotate: Can be any of the following
```
